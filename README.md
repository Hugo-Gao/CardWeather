# 卡片天气

[卡片天气Github地址](https://github.com/Hugo-Gao/CardWeather)

卡片天气是我学年综合实践的作品，这个APP主要集成了聚合天气的API和百度地图的定位API，能够自动定位用户所在地的七日内的天气。
